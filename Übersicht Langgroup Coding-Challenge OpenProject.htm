    Projektbeschreibung

    YOUR TASK

    Your task is to create a library where a user is able to browse through existing books, borrow books and return borrowed books.

    Administrators shall be able to create, update and delete book entries.

    Furthermore administrators should be able to see who has borrowed which book and when, and whether it has been returned (with time stamp)

    Notice: Use PHP with Laravel as framework, vanilla JavaScript or jQuery

    FEATURES

    1. Browse through existing books
    2. Administrators can create new book entries with authors
    3. Administrators can update existing book entries
    4. Administrators can delete existing book entries
    5. Search books
    6. A user must be able to borrow books
    7. A user must be able to bring back borrowed books
    8. Administrators should see which user has borrowed which book and when and whether it has been returned (with time of return) (optional)
    9. Make it user friendly (optional)

    EXAMPLE BOOK:

    Title: "How to be successful"

    Author: John Doe

    Date of publication: 12.06.1998

    REQUIREMENTS:

    This are must have requirements, your application needs to fulfill:

    • PHP >= 8.2
    • Laravel >= 9
    • MySQL (use "migrations" to create the database-tables to follow the "Code-First approach")
    • Vanilla JavaScript and/or JQuery
    • HTML5
    • OpenProject as Project Management Tool
    • GIT (Commit everything to the repository)
    • Supported Browsers: Microsoft Edge, Firefox

    The project should be executable on your machine so that we can do a review together in the follow-up meeting
